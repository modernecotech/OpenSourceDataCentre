# Security Data

Machine-readable security catalogues that are not tied only to cloud software.

- `physical-security-controls.csv` defines physical security controls for commercial-readiness and audit evidence.
- `threat-management-stack.csv` defines the open-source Wiz-like threat-management components and integration surfaces.
- `scanner-coverage.csv` defines source, dependency, image, IaC, Kubernetes, host, network, runtime, compliance, and OT scanning coverage.
