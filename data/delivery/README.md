# Delivery Data

Machine-readable project-delivery controls for moving from reference design to reviewed project.

- `project-gates.csv` defines phase gates and required evidence.
- `authority-permits.csv` tracks approval and permit evidence targets.
- `risk-register.csv` provides a starter project risk register.
- `action-tracker.csv` tracks gap closure and delivery actions.

These files are templates. A real project should assign owners, dates, evidence references, and approval records.
