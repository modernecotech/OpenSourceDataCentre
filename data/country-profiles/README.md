# Country Profiles

This directory contains example planning profiles for country-specific datacentre assumptions.

These files are not official national datasets. Treat every value as a placeholder until it is replaced with measured utility data, local engineering input, procurement quotes, authority requirements, and operator experience.

Each profile covers:

- grid reliability;
- climate and water stress;
- energy and fuel assumptions;
- procurement multipliers;
- sovereignty requirements.

The profile shape is documented in [Country Site Profile Guide](../../docs/deployment/country-site-profile-guide.md).
