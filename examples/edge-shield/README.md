# Edge Shield Examples

These profiles describe example OSDC Edge Shield deployments. They are consumed by the `osdc-edge-config` crate tests so the examples stay valid as the schema grows.

- `3-node-regional-edge.json` is the recommended regional baseline.
- `school-hospital-edge.json` is a small community services profile.
- `ministry-public-service-edge.json` is a public-sector service protection profile.
