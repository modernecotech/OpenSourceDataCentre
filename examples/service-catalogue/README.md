# Service Catalogue Examples

These JSON examples select service bundles for each OSDC deployment scale.

- `50kw-edge-services.json` starts with local identity, edge, backup, observability, and a small cloud core.
- `250kw-regional-pilot-services.json` is the first serious public-sector cloud.
- `1mw-sovereign-cloud-services.json` expands into production cloud, security, developer, data, and operations services.
- `5mw-ai-ready-services.json` adds the national AI and data platform.

The examples are validated by `osdc-models` tests.
