# Data Platform Examples

These examples are starter artifacts for the optional OSDC Data Platform as a Service.

The goal is a Foundry-like open-source workflow:

1. Register a data product.
2. Ingest data from a source system.
3. Land it in an open lakehouse table.
4. Attach ownership, quality checks, lineage, and ontology objects.
5. Expose governed SQL, dashboards, notebooks, apps, and AI context.

The examples are intentionally small. They show the files that the OSDC portal should generate into a Forgejo repository for review and GitOps promotion.
