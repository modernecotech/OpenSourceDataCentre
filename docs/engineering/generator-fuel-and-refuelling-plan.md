# Generator Fuel and Refuelling Plan

Fallback generation is only credible if fuel logistics are credible.

Plan:

- generator rating and runtime target;
- fuel type;
- tank capacity;
- safe storage and bunding;
- refuelling route;
- supplier contract;
- emergency delivery plan;
- fuel polishing or quality testing;
- spill response;
- noise and emissions permits;
- test run cadence.

Fuel access is a site-selection gate, not an afterthought.
