# Liquid Cooling Loop Specification

Liquid loops must be specified as maintainable systems.

Define:

- coolant;
- supply and return temperature;
- pressure and flow range;
- redundancy;
- CDU or heat exchanger;
- filtration;
- water treatment;
- leak detection;
- isolation valves;
- quick disconnects;
- maintenance procedure;
- emergency response.

Loop acceptance requires pressure test, leak test, water-quality baseline, and thermal-load test.
