# Cooling Piping and Instrumentation

This file defines the first P&ID-level evidence target for the rack thermal spine and cooling plant.

Minimum P&ID elements:

- rack rear-door or in-row heat exchanger loop;
- thermal spine supply and return headers;
- pump set with N+1 controls where required;
- strainers, valves, isolation, bypass, drains, vents, and fill points;
- dry cooler, chiller, ground loop, or heat-reuse interface;
- leak detection zones;
- water treatment and filtration points;
- temperature, pressure, flow, conductivity, and leak sensors;
- BMS/EPMS integration points;
- emergency shutdown and fail-open/fail-safe behavior.

Acceptance evidence:

- thermal capacity calculation;
- pressure-drop calculation;
- leak detection test;
- water treatment plan;
- cooling-failover test;
- thermal runaway or ride-through test for high-density racks;
- sensor calibration record.
