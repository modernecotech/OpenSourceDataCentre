# Data Residency Evidence Pack

Data residency needs evidence that data, keys, logs, and backups remain within the approved boundary.

Evidence:

- tenant location policy;
- storage location;
- backup location;
- key custody;
- admin access locations;
- log retention location;
- subprocessors;
- incident notification path;
- export and deletion procedure.

This pack should link to sovereignty docs, customer terms, and audit evidence.
