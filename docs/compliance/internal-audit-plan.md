# Internal Audit Plan

Internal audit checks whether policies and controls are actually operating.

Audit areas:

- access control;
- change management;
- incident management;
- backup and restore;
- supplier risk;
- physical security;
- vulnerability and patch management;
- facility operations;
- sustainability evidence;
- customer SLA evidence.

Each audit should record scope, sample size, findings, owner, due date, and management-review outcome.
