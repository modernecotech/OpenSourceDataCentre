# PCI DSS Physical Scope

This template helps decide whether payment-card workloads or systems create PCI DSS scope.

Questions:

- Are cardholder data environments hosted?
- Which racks, networks, and access paths are in scope?
- Which physical access controls apply?
- Which logs and CCTV evidence are retained?
- Who approves visitor access?
- How are media and failed disks handled?
- How are network segments separated?
- What evidence supports quarterly or annual reviews?

Avoid bringing payment workloads into scope unless the operator is ready for the evidence burden.
