# ISO 27001 ISMS Outline

This is an ISMS readiness outline, not a certification claim.

Minimum ISMS artifacts:

- scope statement;
- interested parties;
- risk assessment method;
- risk register;
- risk treatment plan;
- statement of applicability;
- access-control policy;
- supplier-risk process;
- incident-management process;
- internal-audit plan;
- management-review record;
- continual-improvement log.

OSDC evidence sources:

- security controls catalogue;
- audit evidence register;
- access reviews;
- change records;
- incident records;
- backup/restore evidence;
- vulnerability and patch records.
