# Metering Plan

A credible sustainability report starts with meters.

Required meters:

- utility import/export;
- generator fuel and electrical output;
- PV output;
- BESS charge/discharge;
- DC bus;
- rack or row IT load;
- cooling plant electrical load;
- water use;
- heat reuse where applicable.

Meter data should have time synchronization, calibration records, retention, and exception handling.
