# Egress and Occupant Safety

Datacentre security must not compromise emergency egress.

Review:

- exit route distances;
- emergency lighting;
- exit signage;
- door release logic;
- anti-tailgating device fail-safe behavior;
- muster point;
- visitor/contractor accountability;
- disabled access where required;
- drill cadence.

Life safety takes precedence over asset security during evacuation.
