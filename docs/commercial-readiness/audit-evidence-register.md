# Audit Evidence Register

Commercial trust depends on evidence, not slogans.

Source data:

- [audit-evidence.csv](../../data/commercial/audit-evidence.csv)

Evidence classes:

- design evidence: drawings, calculations, basis of design, third-party review;
- commissioning evidence: L1-L5 tests, defects, retests, signoff;
- operations evidence: MOP/SOP/EOP records, tickets, shift handover, access logs;
- security evidence: access reviews, audit logs, incident records, vulnerability scans;
- compliance evidence: risk register, SoA, internal audit, management review, supplier review;
- sustainability evidence: metered PUE/WUE/CUE, renewable procurement, water use, carbon report.

Evidence status:

- `template`: structure exists, no project data yet.
- `collected`: evidence exists for a specific project.
- `reviewed`: owner review complete.
- `audited`: external audit or authority review complete.

No evidence entry is valid without an owner, date, source system, and retention rule.
