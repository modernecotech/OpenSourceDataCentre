# Site Selection

Site selection determines whether a sovereign datacentre is insurable, maintainable, connectable, and expandable.

Source data:

- [site-selection-scorecard.csv](../../data/site-selection/site-selection-scorecard.csv)

Minimum due diligence areas:

- flood and drainage;
- seismic and structural hazard;
- geotechnical conditions;
- grid and utility capacity;
- fibre route diversity;
- road, fuel, crane, and emergency access;
- noise, environmental, and planning constraints;
- authority having jurisdiction;
- security perimeter and expansion corridor.

Do not freeze a design on cheap land alone. A low-cost site can become expensive if it lacks power, fibre, safe access, drainage, or authority approval.
