# Utility Capacity Checklist

Utility capacity is a primary site-selection gate.

Checklist:

- available grid capacity and upgrade path;
- voltage level and point of connection;
- historical outage rate and voltage stability;
- power-quality constraints;
- protection and metering requirements;
- generator fuel delivery route;
- water availability if the cooling strategy needs water;
- telecom duct access;
- utility permitting timeline;
- cost and lead-time estimate.

Red flag: no credible grid path for the target IT load and fallback strategy.
