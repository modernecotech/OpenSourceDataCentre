# Site Selection Scorecard

Use the scorecard before selecting a pilot or production site.

Source data:

- [site-selection-scorecard.csv](../../data/site-selection/site-selection-scorecard.csv)

Scoring guidance:

- score each criterion from 0 to 5;
- multiply by the weight in the CSV;
- record evidence and reviewer;
- treat any red flag as a design hold until resolved;
- keep the signed scorecard with the project evidence pack.

Minimum acceptance:

- no unresolved critical red flags;
- power and fibre path validated;
- authority and fire review path known;
- climate, water, dust, and corrosion risks understood;
- expansion and logistics paths preserved.
