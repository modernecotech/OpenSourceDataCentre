# GPU Cluster Networking

AI clusters are constrained by network topology as much as GPU count.

Plan:

- 100/200/400/800G fabric options;
- leaf-spine oversubscription;
- storage network separation;
- management network separation;
- tenant isolation;
- telemetry and congestion monitoring;
- optics and cable spares;
- route and fabric failure tests.

Network design should be tied to workload class: inference, fine-tuning, training, HPC, or shared research.
