# High-Density Rack Classes

Source data:

- [high-density-rack-classes.csv](../../data/ai-ready/high-density-rack-classes.csv)

Rack classes should define:

- power range;
- cooling mode;
- network target;
- storage target;
- floor loading;
- leak detection;
- fire/life-safety implications;
- MOP requirement;
- commissioning test.

Do not install high-density AI racks until power, cooling, network, and maintenance evidence match the selected class.
