# AI Tenant Isolation

Shared AI infrastructure needs isolation across compute, storage, network, models, and logs.

Controls:

- namespace/project isolation;
- GPU device allocation policy;
- image signing and scanning;
- network policy;
- storage bucket and volume policy;
- model registry permissions;
- inference endpoint authentication;
- log and trace privacy;
- quota and fair-share enforcement.

Tenant isolation must be tested, not only configured.
