# Generator-Start Test

## Purpose

Prove the fallback generator or rental-generator connection can support the intended critical load during extended outage.

## Preconditions

- Fuel, ventilation, exhaust, earthing, permits, and safety controls verified.
- Load bank or staged IT load available.
- Generator interface and boundary rectifier settings reviewed.
- Fire and electrical safety observers present where required.

## Procedure

1. Record baseline site load and BESS state of charge.
2. Start generator under approved procedure.
3. Transfer or connect through the approved boundary input.
4. Confirm rectifier, DC bus, cooling, telemetry, and alarms.
5. Run for the planned test duration.
6. Return to normal state and inspect for leaks, faults, alarms, and abnormal readings.

## Pass Criteria

- Generator supports planned critical load.
- DC system remains within limits.
- Operators can perform start, transfer, monitoring, and shutdown procedures.
