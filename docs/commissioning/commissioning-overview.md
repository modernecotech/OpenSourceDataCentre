# Commissioning Overview

Commissioning is the proof that the facility, software, operators, and runbooks work together before production launch.

## Levels

| Level | Name | Purpose |
| --- | --- | --- |
| L1 | Component inspection | Confirm delivered parts, labels, certificates, manuals, and installation quality. |
| L2 | Pre-functional checks | Confirm wiring, isolation, settings, fluids, firmware, sensors, and safety prerequisites. |
| L3 | Functional tests | Test individual systems under controlled conditions. |
| L4 | Integrated systems test | Test power, cooling, software, telemetry, and operations together under staged load. |
| L5 | Production readiness | Confirm documentation, spares, training, security, backup, monitoring, and owner sign-off. |

## Required Integrated Tests

## Rule

Do not skip commissioning to save money. A cheap untested datacentre is an expensive outage waiting to happen.
