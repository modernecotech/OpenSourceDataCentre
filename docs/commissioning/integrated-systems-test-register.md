# Integrated Systems Test Register

Integrated systems tests prove that systems work together under realistic failure modes.

Candidate integrated tests:

- grid loss;
- DC bus ride-through;
- generator start;
- cooling failover;
- leak detection;
- fire alarm interaction;
- BESS alarm;
- network carrier failure;
- backup restore;
- access control emergency mode;
- workload shed and recovery.

Integrated tests should link to the sequence of operations and incident procedures.
