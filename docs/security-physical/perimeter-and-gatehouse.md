# Perimeter and Gatehouse

The perimeter is the first commercial security boundary.

Minimum controls:

- fence or equivalent boundary;
- vehicle gate and pedestrian gate;
- lighting;
- visitor identity check;
- delivery check-in;
- guard post or supervised reception where staffing allows;
- emergency access process;
- camera coverage.

Evidence:

- perimeter inspection;
- visitor log;
- incident log;
- access exceptions.
