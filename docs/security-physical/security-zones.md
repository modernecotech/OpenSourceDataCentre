# Security Zones

Define physical zones before access control is designed.

Example zones:

- public approach;
- perimeter and gatehouse;
- reception/visitor area;
- office/support area;
- loading and staging;
- meet-me room;
- data hall;
- customer cage or suite;
- power room;
- cooling plant;
- BESS/generator yard;
- security control room.

Each zone needs:

- authorized roles;
- access method;
- escort rule;
- CCTV coverage;
- logging requirement;
- emergency access path;
- review cadence.
