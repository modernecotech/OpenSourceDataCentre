# CCTV Retention and Review

CCTV supports security and audit evidence but must follow local privacy law.

Define:

- camera coverage map;
- retention period;
- access roles;
- export approval;
- review triggers;
- integrity protection;
- incident evidence handling;
- camera health checks.

Typical review triggers include intrusion, customer dispute, chain-of-custody event, access exception, or incident response.
