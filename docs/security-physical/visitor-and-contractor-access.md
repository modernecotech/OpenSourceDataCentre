# Visitor and Contractor Access

Visitor and contractor access must be pre-approved, logged, escorted, and reviewed.

Required fields:

- identity;
- organization;
- sponsor;
- purpose;
- zones approved;
- escort;
- start and end time;
- equipment carried in or out;
- closeout confirmation.

Contractors performing work need a ticket, MOP or permit-to-work where required, and evidence closeout.
