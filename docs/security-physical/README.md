# Physical Security

Physical security for a commercial or public-sector datacentre is a layered operating model, not just cameras and fences.

Source data:

- [physical-security-controls.csv](../../data/security/physical-security-controls.csv)

Core layers:

- perimeter and vehicle control;
- gatehouse or supervised reception;
- anti-tailgating entry;
- visitor and contractor access;
- CCTV retention and review;
- loading dock and chain of custody;
- rack, cage, and suite access;
- periodic access review;
- break-glass physical access.
