# Critical Components Not to Buy Cheap

Low-cost procurement is useful for commodities, but some systems carry too much safety and continuity risk to buy only on price.

## Require Stronger Review

- Fire detection and suppression.
- Electrical protection, breakers, fuses, switchgear, busbars, and earthing systems.
- Battery modules, BMS, DC converters, rectifiers, and high-energy connectors.
- Generators, fuel systems, and transfer equipment.
- Pumps, valves, pressure components, and liquid-cooling manifolds.
- Access control, CCTV/NVR, and audit-critical security systems.
- Network core, storage, and backup media for production services.

## Procurement Rule

For these components, require documented standards, local code compliance, spares, manuals, support path, commissioning test, and engineering approval before purchase.
