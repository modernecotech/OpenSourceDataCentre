# Second-Source Requirements

Every critical BOM line needs a second-source plan or a documented local substitution path.

## Required Fields

- Component function.
- Criticality: critical, important, optional, or future.
- Primary supplier.
- Second-source supplier or local substitute.
- Interface standard.
- Lead time.
- Local spare classification.
- Failure symptoms.
- Replacement procedure.
- Post-replacement commissioning test.

## Rules

- Avoid proprietary interfaces where a documented open interface is available.
- Do not buy a critical component that only one foreign technician can restart.
- Prefer field-replaceable units with clear manuals and local support.
- Keep critical spares on site when lead time exceeds acceptable downtime.
- Review second-source plans after each procurement cycle.
