# Capacity Management

Capacity management links sales, engineering, and operations.

Track:

- rack space;
- contracted kW;
- measured kW;
- cooling capacity;
- network ports;
- cross-connect capacity;
- storage and cloud capacity;
- generator and BESS autonomy;
- spares.

Capacity changes should require evidence from DCIM, EPMS, BMS, and commercial contracts.
