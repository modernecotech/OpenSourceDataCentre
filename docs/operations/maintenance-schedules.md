# Maintenance Schedules

Maintenance schedules should be simple, visible, and tied to evidence.

## Schedule Categories

- Daily: alarms, dashboards, access logs, environmental readings, backup status, and queue health.
- Weekly: filter checks, visual inspections, spares inventory changes, patch notes, and failed-job review.
- Monthly: generator exercise, backup restore sample, cooling failover check, security access review, and capacity review.
- Quarterly: DC bus ride-through test where safe, battery health review, firmware review, fire-system coordination, and incident drill.
- Annual: integrated systems test, disaster-recovery exercise, insurance/code review, spares reset, and training refresh.

## Records

Each task should record owner, date, result, measured values, defects, corrective action, and next due date.
