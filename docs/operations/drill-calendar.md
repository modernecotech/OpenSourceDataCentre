# Drill Calendar

Drills prove that procedures work before emergencies.

Recommended drills:

- grid loss;
- generator start failure;
- cooling failure;
- leak detection;
- fire alarm evacuation;
- BESS alarm;
- cyber incident;
- physical intrusion;
- backup restore;
- customer communication exercise.

Each drill should record scope, participants, results, defects, and corrective actions.
