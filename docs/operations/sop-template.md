# Standard Operating Procedure Template

Use an SOP for routine repeatable work.

Required fields:

- purpose;
- scope;
- roles;
- normal frequency;
- prechecks;
- procedure;
- expected result;
- exceptions;
- escalation;
- evidence to retain;
- review cadence.

Examples:

- daily facility walkdown;
- weekly generator inspection;
- monthly access review;
- quarterly backup restore test;
- rack installation checklist.
