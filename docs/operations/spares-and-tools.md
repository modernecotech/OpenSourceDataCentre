# Spares and Tools

Local spares are part of sovereignty. The site should not wait weeks for a common failure part.

## Starter Spares

- Fans, filters, sensors, valves, pumps, power supplies, NICs, drives, fibre optics, patch leads, optics, common fasteners, and labels.
- DC protection spares approved by the electrical engineer.
- Rack blanking panels, shelves, rails, and cable-management hardware.
- Backup media and tested restore hardware.
- SBC gateways and storage for offline recovery tools.

## Tooling

- Multimeters, clamp meters, thermal camera, label printer, torque tools, fibre tools, laptop console kit, serial adapters, and network test gear.
- PPE and lockout/tagout equipment.
- Printed emergency runbooks and as-built diagrams.

## Records

- Minimum stock.
- Reorder point.
- Lead time.
- Storage conditions.
- Expiry date where relevant.
- Compatible component list.
