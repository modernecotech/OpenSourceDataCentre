# Method of Procedure Template

Use a MOP for planned work that can affect availability, safety, security, tenant service, or audit evidence.

Required fields:

- change ID;
- title;
- owner;
- affected systems;
- maintenance window;
- risk level;
- prerequisites;
- tools and spares;
- step-by-step method;
- hold points;
- rollback plan;
- communications plan;
- safety controls;
- evidence required;
- approval signatures;
- closeout notes.

No high-risk facility or platform change should proceed without an approved MOP.
