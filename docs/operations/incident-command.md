# Incident Command

Commercial operations need a clear incident command model.

Roles:

- incident commander;
- facilities lead;
- network/platform lead;
- security lead;
- communications lead;
- customer liaison;
- safety officer where required.

Incident phases:

1. Declare.
2. Stabilize safety and service.
3. Assign roles.
4. Preserve evidence.
5. Communicate status.
6. Restore service.
7. Verify recovery.
8. Run post-incident review.

The incident commander controls emergency access, escalations, and customer-impacting communications until handover or closure.
