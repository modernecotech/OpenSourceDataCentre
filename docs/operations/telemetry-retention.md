# Telemetry Retention

Telemetry retention supports operations, SLA evidence, audits, and sustainability reporting.

Retention classes:

- high-resolution operational telemetry;
- alarm and event logs;
- security logs;
- access logs;
- energy and sustainability metrics;
- customer SLA evidence;
- incident evidence.

Each class needs retention duration, storage location, access policy, privacy rule, and deletion process.
