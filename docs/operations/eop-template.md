# Emergency Operating Procedure Template

Use an EOP for abnormal or emergency conditions.

Required fields:

- emergency condition;
- immediate safety action;
- incident commander;
- communication channel;
- automatic systems expected to operate;
- manual actions allowed;
- actions forbidden without approval;
- escalation path;
- recovery criteria;
- evidence to preserve;
- post-incident review requirement.

Example EOPs:

- grid loss;
- cooling failure;
- BESS alarm;
- fire alarm;
- water leak;
- physical intrusion;
- cyber incident affecting facility controls.
