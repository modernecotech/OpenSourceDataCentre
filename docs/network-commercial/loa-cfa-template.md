# LOA/CFA Template

Letter of authorization and connecting facility assignment records authorize a cross-connect.

Template fields:

- customer name;
- carrier or peer name;
- service order;
- A-side demarcation;
- Z-side demarcation;
- media type;
- connector type;
- speed;
- authorized contact;
- expiry date;
- installation notes;
- acceptance signature.

No cross-connect should be installed without authorization and demarcation evidence.
