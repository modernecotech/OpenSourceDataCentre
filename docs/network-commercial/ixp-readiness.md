# IXP Readiness

An internet exchange point can turn a sovereign datacentre into national digital infrastructure.

Readiness requirements:

- carrier-neutral meet-me room;
- redundant switching fabric;
- route-server policy;
- participant onboarding;
- port product definition;
- abuse handling;
- traffic statistics;
- maintenance windows;
- route filtering and security controls;
- governance model.

The first OSDC milestone should model IXP readiness as a checklist before attempting to operate an exchange.
