# Fibre Route Diversity

Route diversity protects against civil works, cable cuts, and carrier outages.

Evidence:

- carrier route maps;
- duct entry points;
- handhole and manhole records;
- diverse building entries;
- meet-me room path;
- repair SLA;
- splicing and patch records;
- no shared single point of failure where diversity is claimed.

Never sell route diversity without proof of physical diversity.
