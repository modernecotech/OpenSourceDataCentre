# Remote Hands SLA

Remote hands SLA should define response target, not guaranteed repair.

Scope:

- visual inspection;
- power cycle;
- cable patch;
- media swap;
- smart hands by approved MOP;
- emergency hands by incident commander.

Exclusions:

- unsafe work;
- missing authorization;
- unavailable spares;
- work outside approved demarcation;
- work requiring licensed trades.
