# Price Book Template

The price book turns service definitions into transparent cost recovery.

Product families:

- rack unit;
- half rack;
- full cabinet;
- private cage;
- contracted kW;
- cross-connect;
- IP transit or IXP port;
- access badge;
- remote hands;
- smart hands;
- shipping and receiving;
- cloud platform services.

Each price line should include unit, setup fee, recurring fee, included support, SLA class, and evidence source for billing.
