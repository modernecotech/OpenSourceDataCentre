# Customer Demarcation and Responsibility Matrix

This file extends the commercial-readiness responsibility matrix into service-order detail.

Demarcation examples:

- rack power: facility PDU outlet or whips to customer PDU;
- cross-connect: meet-me panel port to customer panel port;
- cloud service: OSDC API and platform boundary to tenant workload;
- remote hands: task scope in approved ticket;
- physical access: security boundary to customer assigned space.

Every service order should record the exact demarcation point and evidence owner.
